This folder contains call log data for the first quarter of 2025.

File Descriptions:
- calls_jan.csv: Call records for January 2025
- calls_feb.csv: Call records for February 2025
- calls_mar.csv: Call records for March 2025

Columns:
- caller_id: ID of the person who made the call
- receiver_id: ID of the person who received the call
- duration_sec: Duration of the call in seconds
- timestamp: Date and time of the call
- location: Location from where the call was made

Use these files to test HDFS file system operations.
